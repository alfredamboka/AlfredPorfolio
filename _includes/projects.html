Automated Soil Desalinator

Python programming GIS Remote sensing IOT Precison Surveying

This project is aimed at improving agricultural land that is saline (has too much mineral salts). Such land cannot support growth of several crops thus the need to lower salt level. Using GIS, we map the areas with such soil properties and embed GIS system to platform(tractor and other farm ploughing equipment) and automatically quantify amount of solution to be used. The system is automated using robotics technologies and some few manual work can be performed.
